Front End

Desenvolvimento web front-end é o desenvolvimento da interface gráfica do usuário de um site, através do uso de HTML, CSS e JavaScript, para que os usuários possam visualizar e interagir com esse site.

Site One Page

Ideal para negócios de pequeno ou médio porte o site One Page é um site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única página.

Wireframe

Wireframe é uma representação visual da estrutura básica de uma página. Ele deve ser feito de maneira mais simples possível, mostrando apenas o essencial, como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta página.
